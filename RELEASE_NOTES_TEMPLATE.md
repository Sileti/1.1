## ğŸ“– Getting Started

### Basic Usage
1. Enter or paste text in the text area
2. Select a voice from the dropdown menu
3. Choose your preferred playback speed
4. Click **Play** to generate and play speech

### Global Hotkey (Clipboard Reading)
1. Copy any text to clipboard (`Ctrl+C`)
2. Press **Ctrl+Alt+S** (or your custom hotkey) from anywhere in Windows
3. Sileti will automatically speak the clipboard contents

### Customizing the Hotkey
1. Right-click the system tray icon
2. Select "Change Hotkey"
3. Press your desired key combination
4. Click **Save**

### System Tray Features
- Closing the window minimizes to system tray
- Right-click tray icon for quick access menu (Show, Speak Clipboard, Play/Pause, Quit)
- Double-click tray icon to restore window
- Control playback and settings without opening the main window
